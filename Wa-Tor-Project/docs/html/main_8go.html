Wa-Tor Simulation in Go.

This file implements the Wa-Tor predator-prey simulation. The simulation contains fish and sharks on a toroidal grid. Each chronon (time step) updates the world according to the rules:

  • Fish move and reproduce
  • Sharks move, hunt fish, reproduce, and starve
